Bonus Part for some JavaScript Questions

How does JavaScript work?

JavaScript is a programming language for interacting with the browser's interface. The original code is run through a compiler software, which converts it to byte code that the machine can understand and execute. JavaScript, on the other hand, does not require any compilation. Instead, a browser interpreter examines the JavaScript code, understands each line, and then executes it.

What does event loop do?

The Event Loop is a feature of engines like V8 that allows JS to outsource jobs to other threads. Long-running tasks are executed independently from the main JS thread by the browser and Node APIs, which then enqueue a callback function to run on the main thread after the job is completed. Because it should only ever wait for synchronous code from your JS functions, JS is considered non-blocking. Consider the Event Loop to be a message queue between a single JS thread and the operating system.
